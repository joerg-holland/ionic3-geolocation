## How to build a Ionic-Geolocation app?

This is a Ionic-Geolocation template.

## Requirements
1. Node.js (v6.11.2 LTS) [https://nodejs.org/](https://nodejs.org/)
  ```bash
  // Check successful installation with:
  $ node --version
  ```
2. NPM (v5.3.0)
  ```bash
  $ sudo npm install -g npm@5.3.0
  ```
  ```bash
  // Check successful installation with:
  $ npm --version
  ```
3. Apache Cordova (v7.0.1)
  ```bash
  $ sudo npm install -g cordova@7.0.1
  ```
  ```bash
  // Check successful installation with:
  $ cordova --version
  ```
4. Ionic CLI (v3.9.2)
  ```bash
  $ sudo npm install -g ionic@3.9.2
  ```
  ```bash
  // Check successful installation with:
  $ ionic --version
  ```

## How to use this template?

To use this template, either create a new ionic project using the ionic node.js utility and follow the tutorial below, or copy the files from this repository [Ionic-Geolocation](https://github.com/jschax/ionic-geolocation).

Then, to run it, cd into the project `ionic-geolocation` and run:

1. Install the packages:
  ```bash
  $ npm install
  ```
2. Build the project:
  ```bash
  $ npm run build
  ```
3. Build the platform:
  ```bash
  $ cordova platform add android
  ```
4. Build the app:
  ```bash
  $ cordova build android
  ```
5. Deploy the app to device:
  ```bash
  $ cordova run android --device
  ```

Substitute `android` for `ios` if on a Mac.
